# Automated Nutrition Analysis of Recipes Using NLP

## Overview
This project is a web-based nutrition analysis system developed using Python and Natural Language Processing (NLP). The application extracts ingredients from recipes and calculates nutritional values such as calories, protein, fat, and carbohydrates.

## Features
- Ingredient extraction using NLP
- Nutrition calculation
- Calorie analysis
- Food dataset integration
- Flask-based web application

## Technologies Used
- Python
- Flask
- Pandas
- NLP
- HTML/CSS

## Files
- app.py → Main Flask application
- preprocess.py → Text preprocessing and NLP
- merged_dataset.csv → Nutrition dataset
- index.html → User interface
